# aslezak.com
source code and graphics for aslezak.com

### Tools
- Gatsby
- React
- Rebass
- Emotion
- styled-system
- Formik
- AWS Lambda
- AWS Route53
- Mailgun
- Netlify
- Sketch
- Sass

###  Dev

`gatsby develop -H 0.0.0.0` 

### Build

```
`gatsby build` // build for production
`gatsby serve`  // test production build locally
```



Deploy
```
cd public
netlify deploy
```
